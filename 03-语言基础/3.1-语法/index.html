3.1 区分大小写


ECMAScript中的一切区分大小写（变量名，函数，操作符）

3.2 标识符


标识符:变量、函数、属性或函数参数的名称
  1. 第一个字符必须是字母，下划线_或美元符号$
  2. 剩下的字符可以说字母，下划线，美元符号和数字
  3. 关键字，保留字，true，false和null不能作为标识符

3.3 注释

// 单行注释
/**/ 块注释

3.4 严格模式

ES 5引入严格模式的概念，是一种不同的Javascript解析和执行模型，对于不安全的活动会抛出错误
使用方法
"user strict"
function dosomething{ "user strict" }

3.5 语句

语句结尾加入;有助于防止省略造成的问题
使用C语言代码块{}让内容更清晰，减少出错
